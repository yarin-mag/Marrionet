# Marionette — Feature Ideas

## Bigger Bets

**8. Cost budget alerts**
Set a per-agent token/cost budget. Turn the card red when exceeded. Gives safety rails on expensive autonomous runs.

**9. Side-by-side agent comparison**
When 4 agents run in parallel on different approaches, you want to compare them. A split view or table mode showing all their current tasks + token usage at once.

**10. Mobile-friendly view**
The dashboard is desktop-only right now. A stripped-down mobile view showing just agent status + current task would be genuinely useful.
